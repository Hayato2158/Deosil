## 位置づけ（重要）

本ツールは **「勤怠管理システム」ではなく「勤怠管理補助ツール」** です。

以下を目的としていません。

- 正式な勤怠記録
- 法令対応（36協定、残業管理、深夜手当 等）
- 給与計算・労務管理

これらを **保証・代替するものではありません**。

## データの正確性について（免責）

本ツールで記録・表示される勤怠データについて、  
**正確性・完全性・信頼性を保証するものではありません**。

以下のような事象が発生する可能性があります。

- 打刻忘れ・誤操作
- 端末・ブラウザ依存による不整合
- 通信障害・同期遅延
- 実働時間・差分計算の誤差

本ツールの利用によって生じたいかなる損害についても、  
**開発者は一切の責任を負いません**。

## 個人情報の取り扱いについて（重要）

### 勤怠情報は個人情報に該当します

本ツールで扱う以下の情報は、  
**特定の個人を識別可能な個人情報に該当します**。

- メールアドレス
- メールアドレスに紐づく勤怠情報  
  （出勤時刻・退勤時刻・勤務日・勤務時間 等）

利用者は、  
**これらの情報が開発者（私）に知られる可能性があることを理解した上で利用してください。**

---

## Supabase 利用に伴うリスクについて

本ツールでは **Supabase** を利用して以下を行います。

- メールアドレスによる認証
- 勤怠データの保存・取得

そのため、以下のリスクが存在します。

- Supabase 側の障害・設定不備
- 認証情報・勤怠データの漏洩
- 不正アクセス・第三者による情報取得

**メールアドレスおよびそれに紐づく勤怠情報が、  
Supabase 経由で流出するリスクを完全に排除することはできません。**

これらのリスクを理解・同意した上で利用してください。
